#testing 

Report about end of a test activity, e.g. [[testing phases|test phase]].

If [[test end criteria]] are reached, the [[test manager]] created a test summary report based on the last [[test progress report]]s and other information.

Content:
- summary of testing performed
- deviation from planned testing
- test completion evaluation
- factors that blocked progress
- test measures
- residual [[risk]]
- test deliverables
- reusable test assets
- lessons learned
