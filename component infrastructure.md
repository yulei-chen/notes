#se 

- Components: Components that can be composed, connected, and deployed.
- Repository: Private and/or public archive of components. Usually used to offer, search for, and distribute external components.
- Component Framework: Provides a framework for deploying, creating, composing, distributing, and connecting components. Often also referred to as middleware.
- Supporting Tool: Development environment for component-based systems, with views for component development, component archive, component composition, component deployment, etc.
- Platform: Underlying execution environment, such as a computer with an operating system, database system, and network access.
