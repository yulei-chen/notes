#testing 


Types:

- **Definitional use**, `def(r)`
	- Value assignments, state changing
	- e.g., `r = m` or `r = 5`
- **Computational use**, `c-use(m,n)`
	- Usage in statements, state preserving
	- e.g., `r = m` % `n or r = op1(m,n)`
- **Predicative use**, p-use(r)
	- Usage in decisions, state preserving
	- e.g., `while(r != 0)` or `if(r != 0)`
